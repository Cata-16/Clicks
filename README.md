# Clicks

Este proyecto es una actividad interactiva tipo clicker, creada con Three.js.  
El usuario acumula puntos haciendo clic en la pantalla y puede comprar mejoras que aumentan su puntaje o su velocidad de generación.  
Mientras tanto, un cubo 3D rota continuamente como elemento visual dinámico de fondo.

# Tecnologías utilizadas

- HTML 
- CSS  
- JavaScript
- [Three.js]
